It's very simple
All you need to have is the following 3 files
      JQuery (jquery.min.js)
      Bootstrap JS (bootstrap.min.js)
      Bootstrap CSS (bootstrap.min.css)

and following meta tags
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">


<script type="text/javascript" src="jquery.min.js"></script>
<script type="text/javascript" src="bootstrap.min.js"></script>
<link rel="stylesheet" type="text/css"href="bootstrap.min.css"/>

In simple terms the content in the webpage should adapt the screen irrespective of the device.

Modal title

Entrie bootstrap components follows bootstrap gridsystem.
What it does is it divides the entire screen width into 12 equal coloumns
You can use the following classes to achieve this behaviour
col-md-*
col-sm-*
col-lg-*
col-xs-*
 All these are based on media queries. (css3 feature)
